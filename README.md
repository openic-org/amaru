# Project Amaru
Main repository for Project Amaru.

## File Structure

| Folder  | Subfolder                   | Description    |
|:-------:|:----------------------------|:---------------|
| analog  | core <br> pads <br> tb      | core blocks <br> pad-related blocks <br> testbenches  |
| digital | verilog <br> synth <br> pnr | verilog flies <br> synthesis-related files <br> place-and-route-related files |
| python  | code <br> lib <br> tb       | code files <br> function libraries <br> testbenches |
| simulations  | ngspice <br> verilog       | verilog simulation results <br> verilog simulation results |

## File Types

| File    | Description                 |
|:-------:|:----------------------------|
| `.sch`  | text |
| `.sym`  | text |
| `.py`   | text |
